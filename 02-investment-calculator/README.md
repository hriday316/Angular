
# Basic Investment Calculator

This repository contains a small Angular application that calculates investment results.

- **Framework:** Angular (built with the Angular CLI)
- **App location:** `src/` (standard Angular project structure)

- **Language:** TypeScript

Quick start:

```bash
npm install
ng serve --open
```

Build for production:

```bash
ng build --prod
```

Feel free to edit the source under `src/app/` and open an issue or PR on GitHub.
